## 説明

## 注意

大学非公式です。責任は負いかねます。

## 開発環境

OS: MacOS Catalina 10.15.2~  
Python: 3.6.9  

仮想環境: Pipenv  
実行例:

```sh
pipenv shell

>>> exit
```
